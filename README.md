# pf-music-school-lp

ピアノ教室の体験レッスン申込を想定した **1ページ完結のLP** です。  
教材（HTML/CSS/JavaScript）をベースに、文言・画像差し替えとレイアウト調整を行い、LPとして再構成しました。

## Demo（GitHub Pages）
- 公開URL：`https://sasaki-yuuki37.github.io/pf-music-school-lp/`

## Features / 工夫した点
- 実際のピアノ教室の「集客の流れ（興味→理由→体験→申込）」を意識して、セクション順・導線を設計
- 元教材は**プログラミングスクールの複数ページ構成**だったものを、**ピアノ教室集客用の1ページLP**に再設計
  - 文言・画像を全面的に差し替え（サービス内容が伝わるようにコピーも調整）
- 教材デザインよりも**シンプルで落ち着いたトーン**に調整  
  - clip-path など装飾が強い表現は使わず、余白・文字組・配色で整える方針に変更
- 画像を `object-fit: cover` でトリミングし、見た目の統一感を確保
- favicon / apple-touch-icon を追加
