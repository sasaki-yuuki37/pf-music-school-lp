# pf-music-school-lp

ピアノ教室の体験レッスン申込を想定した **LP** です。  
教材の複数ページ構成のデザインをベースに、1ページLPとして再構成しました。

内容はピアノ教室向けに、文言・画像の差し替えとレイアウト調整を行い、スマホ表示も含めて閲覧しやすいようにレスポンシブ対応しています。

## Features / 作り変えた際のポイント
- 実際のピアノ教室の「集客の流れ（興味→理由→体験→申込）」を意識して、セクション順・導線を設計
- 教材デザインよりも**シンプルで落ち着いたトーン**に調整  
  - clip-path など装飾が強い表現は使わず、余白・文字組・配色で整える方針に変更
- スマホでの可読性を優先し、文字サイズ・余白・ボタン配置を微調整
